# Analyst Agent (RSS -> Groq -> CSV)

## Setup
1. Copy `.env.example` -> `.env` and put your GROQ API key and URL.
2. Create and activate virtualenv:
   - macOS / Linux:
     ```
     python3 -m venv venv
     source venv/bin/activate
     ```
   - Windows (PowerShell):
     ```
     python -m venv venv
     venv\Scripts\Activate.ps1
     ```
